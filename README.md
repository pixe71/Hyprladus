## Before Installation
No requirements needed but it's recommended to make a snapshot in case the installation fail.
<br>

## Installation
```shell
# Clone Hyprladus repository
git clone https://github.com/sachatrouve1/Hyprladus.git

cd Hyprladus

# Launch installation script and let the magic do the rest
./installer.sh
```
<br>

## Utility
Global theme is generated by Pywal, based on the current wallpaper.
To switch wallpaper with its theme, you can use the `set_wallpaper.sh` script with its alias command as follow:
```setwallpaper path/wallpaper.png```. Or use the option `Set wallpaper theme` on the selected image file within Thunar
<br>

## Things to know - ALL the shortcuts
### Window & Session Management
| Shortcut               | Action                                 |
| ---------------------- | -------------------------------------- |
| `Super + Q`            | Close focused window                   |
| `Alt + F4`             | Force kill focused window              |
| `Super + Alt + Delete` | Exit Hyprland session                  |
| `Super + W`            | Toggle between tiled and floating mode |
| `Super + G`            | Toggle group mode for focused window   |
| `Super + F12`          | Maximize active window                 |
| `Super + Delete`       | Lock the screen (`hyprlock`)           |
| `Super + Shift + F`    | Pin / Unpin the focused window         |
| `Super + Backspace`    | Open power menu (Rofi)                 |
| `Ctrl + Alt + W`       | Reload or toggle Waybar                |

### Applications
| Shortcut             | Action                             |
| -------------------- | ---------------------------------- |
| `Super + Enter`      | Open terminal (`kitty`)            |
| `Super + E`          | Open file manager (`thunar`)       |
| `Super + C`          | Launch text editor (`nvim`)        |
| `Super + F`          | Launch web browser (`firefox`)     |
| `Ctrl + Shift + Esc` | Open system monitor (`btop`)       |
| `Super + D`          | Launch Discord                     |
| `Super + M`          | Launch Spotify                     |
| `Super + S`          | Launch Steam                       |
| `Ctrl + Esc`         | Launch color picker (`hyprpicker`) |
| `Ctrl + ²`           | Launch screen ruler (`kruler`)     |

### Rofi Menus
| Shortcut      | Action               |
| ------------- | -------------------- |
| `Super + A`   | Application launcher |
| `Super + Tab` | Window switcher      |
| `Super + V`   | Clipboard manager    |

### Window Navigation
| Shortcut                | Action                     |
| ----------------------- | -------------------------- |
| `Super + ← / → / ↑ / ↓` | Move focus between windows |
| `Super + H / J / K / L` | Move focus (Vim-style)     |

### Worskpaces
| Shortcut                     | Action                                 |
| ---------------------------- | -------------------------------------- |
| `Super + [1–10]`             | Switch to workspace                    |
| `Super + Ctrl + → / ←`       | Switch to next/previous workspace      |
| `Alt + Tab`                  | Switch to previous workspace           |
| `Super + Ctrl + ↓`           | Jump to first empty workspace          |
| `Super + Shift + [1–10]`     | Move window to workspace               |
| `Super + Ctrl + Alt + → / ←` | Move window to next/previous workspace |
| `Super + Mouse Scroll`       | Cycle through workspaces               |
| `Super + Alt + ²`            | Move window to scratchpad              |
| `Super + ²`                  | Toggle scratchpad workspace            |


### Move & resize windows
| Shortcut                               | Action                   |
| -------------------------------------- | ------------------------ |
| `Super + Shift + ↑ / ↓ / ← / →`        | Resize focused window    |
| `Super + Ctrl + Shift + ↑ / ↓ / ← / →` | Move focused window      |
| `Super + Z`                            | Move window with mouse   |
| `Super + X`                            | Resize window with mouse |

### Screenshots
| Shortcut        | Action                     |
| --------------- | -------------------------- |
| `Print`         | Screenshot selected region |
| `Shift + Print` | Screenshot entire screen   |

### Miscellaneous
| Shortcut            | Action                  |
| ------------------- | ----------------------- |
| `Super + O`         | Make window opaque (deppends on `windowrule`     |
| `Super + Shift + O` | Make window transparent (deppends on `windowrule`|
| `Super + = / )`     | Zoom cursor in/out      |

### Audio & Media control
| Shortcut                                      | Action                      |
| --------------------------------------------- | --------------------------- |
| `XF86AudioMute`                               | Toggle output mute          |
| `XF86AudioMicMute`                            | Toggle microphone mute      |
| `XF86AudioLowerVolume / XF86AudioRaiseVolume` | Decrease / Increase volume  |
| `XF86AudioPrev / XF86AudioNext`               | Previous / Next media track |
| `XF86AudioPlay / XF86AudioPause`              | Play / Pause media          |

### Brightness
| Shortcut                | Action              |
| ----------------------- | ------------------- |
| `XF86MonBrightnessUp`   | Increase brightness |
| `XF86MonBrightnessDown` | Decrease brightness |

### Extra Functions
| Shortcut        | Action                                 |
| --------------- | -------------------------------------- |
| `XF86Display`   | Open display settings (`nwg-displays`) |
| `XF86Tools`     | Open Hyprland config in `kitty`        |
| `XF86Favorites` | System info via `fastfetch`            |
| `XF86Keyboard`  | Switch keyboard layout                 |
